Slack-Hero (stealth Mode)

After making a few small apps, and a couple bigger ones, I wanted to re-create an app that I used every day, namely Slack. Taking one of my existing starter projects I was able to get the sidebar with navigation running pretty easily. The Slack API was nicely documented so it was possible to dynamically populate Channels, DM's and group DM's that I was already using. Before I knew it, I had an "Electron-Hero" version of Slack that I was able to actually use in my day to day work.


After I used the standard version of Slack-Hero, I started to think about what cool things I could add. While using Slack, many times we'll post documents or lists of user ids, etc. that we want to refer to later. Pinning a Slack entry is nice, but I always thought it would be cool to be able to have a list of "wikis" that would allow of maintaining pages in one central location. Since I'm storing the wiki entries as just regular Slack posts, each channel can have their own collection of wikis that's easy to reference via the tab.

The Wiki entries shows up as a list with a heading and subheading. When you click on one of them it opens another window to show you the details. Since we're using the built in security of Slack, only the user who posted the Wiki can edit it, but anyone can view it.


The other cool thing that we thought would make a great addition to Slack would be a team calendar for each channel. This way, important events within the team (such as UAT testing, days off, etc.) could be put on the calendar so everyone can keep track of them.

Another great custom feature you can see here is a "segmented list", which contains short cuts for your most frequently used channels. These allow you to close the sidebar completely, yet still being able to get to the channels you spend time in quickly. Best of all, it was implemented with just two additional files, one JavaScript and one CSS file.
