Start at the beginning

Electron-Hero is based on the premise of having a "run-time engine" that is used to run Electron-Hero apps. These apps are virtually identical to a traditional Electron app but the code is loaded and run dynamically. Any Electron app runtime has a lot of overhead. If you are running 3 Electron apps you've got this overhead 3 times. Similarly, if you have a few Electron apps running that are using the same node modules, you could have a lot of duplicated node modules as well. Electron-Hero is an attempt to make this whole scenario more efficient.

THE CONSOLE
 

The Electron-Hero console is the heart of it all. Using the very simple console, you can put the URL in for a slack app and then hit "Install". Once installed, you simply click "Run" and you're up and running. See below some of the apps we've developed as part of developing Electron-Hero. Some are simple, some are more complex, it's all a work in progress, so consider all these apps BETA.

THE APPS
 

Starter https://github.com/electron-hero-apps/starter

The "starter" app is about as simple of an Electron-Hero that you can get. It's a simple 2 pane app using PhotonKit to give it some UI love. If you want to see what makes Electron-Hero tick, this is a great place to start. Load it up, and then go mess with the source code and just hit "Reload" and you'll see your changes!
